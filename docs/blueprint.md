# **App Name**: Eracov Properties

## Core Features:

- Tenant Profiles: Create and manage tenant profiles with contact information and lease details.
- Lease Tracking: Track lease agreements, payment statuses, and lease expiration dates.
- Payment Reminders: Automatically generate and send payment reminders to tenants.
- Maintenance Requests: Log, track, and manage maintenance requests from tenants.
- Automated Responses: Employ an LLM to formulate draft replies to tenants regarding maintenance requests; let the LLM tool determine when outside resources should be contacted to help form the draft.
- Property Details: View property details, including address, property type, and other relevant information for Grand Midtown, Midtown, and Grand Midtown Annex.
- Document Storage: Store and manage important documents such as lease agreements and inspection reports.
- Client Property Management: Manage properties for clients who have signed a tenancy agreement.
- Airbnb Monitoring: Monitor units used as Airbnb rentals.

## Style Guidelines:

- Primary color: Steel blue (#4682B4) for reliability and professionalism.
- Background color: Light blue (#E6F0FF), a very desaturated variant of the primary hue.
- Accent color: Light sky blue (#87CEEB) to add emphasis without overpowering the calm aesthetic.
- Body and headline font: 'PT Sans', a humanist sans-serif font.
- Use clean and professional icons for navigation and key features.
- Maintain a clean and organized layout to ensure ease of navigation.
- Use subtle transitions to enhance user experience.
